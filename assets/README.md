# assets/

This directory is for static assets used by the app and documentation.

## demo.gif

Place your demo GIF here as `assets/demo.gif`.

**What to record:**
1. Open the app (show the Emo title and empty chat)
2. Type a message in the input ("I'm having a tough day")
3. Tap Send — show the loading spinner
4. Show the empathetic bot reply appearing
5. Tap the menu icon to open the Saved Messages drawer
6. Optionally: tap the delete icon on a saved message

**Tips:**
- Keep it 5–10 seconds max, looping
- Use a clean emulator or device (hide notifications)
- Record at 1x scale — no need for HD, clarity matters more than resolution
- Tools: Android Studio screen recorder, Xcode Simulator recorder, OBS, or `scrcpy`

The README references this file as `![Demo](assets/demo.gif)`.
